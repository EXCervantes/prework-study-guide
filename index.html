HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • Semantic HTML elements are elements that describe its content and its role on the webpage.
  • Content is inserted between opening and closing tags.
  • Tag refers only to what appears inside the angle brackets.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Use a comma to combine multiple selectors into a single CSS rule.
  • The asterisk selector applies a rule to all the elements visible on the page.
  • Display property specifies the display behavior of an element.
  • A link element creates the connection to the external file.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add: adds modifications in the current working branch to the staging area
  • git pull: receives a branch's changes into the local environment
  • git origin: indicates the source of the pull will be in the GitHub repo
  • git -b: creates a new branch
