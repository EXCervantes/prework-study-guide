HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • Semantic HTML elements are elements that describe its content and its role on the webpage.
  • Content is inserted between opening and closing tags.
  • Tag refers only to what appears inside the angle brackets.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add: adds modifications in the current working branch to the staging area
  • git pull: receives a branch's changes into the local environment
  • git origin: indicates the source of the pull will be in the GitHub repo
  • git -b: creates a new branch
